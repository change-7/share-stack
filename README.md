# Share Stack

안드로이드의 여러 앱에서 공유한 텍스트, 이미지, 동영상, PDF와 일반 파일을 하나의 스택에 모아두었다가 필요한 항목을 선택해 다시 공유하는 앱입니다.

예를 들어 A 앱에서 사진을, B 앱에서 파일과 텍스트를 Share Stack으로 보내면 항목들이 앱 안에 누적됩니다. 이후 여러 항목을 선택해 다른 앱으로 한 번에 공유할 수 있습니다.

## 주요 기능

- 텍스트, 이미지, 동영상, PDF, 일반 파일 수집
- 여러 앱에서 받은 항목을 하나의 스택으로 누적
- 동일한 내용의 중복 항목 자동 제외
- 이미지·동영상·PDF 썸네일 미리보기
- 카드 보기와 목록 보기 즉시 전환
- 선택 공유, 텍스트 복사, 선택 삭제
- 앱 아이콘과 알림에 현재 스택 개수 표시
- 공유받은 뒤 앱을 자동으로 열지 않고 백그라운드에서 수집
- 필요하면 설정에서 공유받은 뒤 앱 열기 활성화
- 공유 후 스택 자동 비우기 설정

## 사용 방법

1. 다른 앱에서 텍스트, 사진 또는 파일의 **공유** 메뉴를 엽니다.
2. 공유 대상에서 **Share Stack**을 선택합니다.
3. 다른 앱에서도 같은 방법으로 항목을 계속 추가합니다.
4. Share Stack을 열고 보낼 항목을 선택합니다.
5. **선택공유**를 누르고 대상 앱을 선택합니다.

## 파일과 텍스트 공유 방식

수신 앱과의 호환성을 높이기 위해 파일과 텍스트를 분리해서 처리합니다.

- 사진과 파일은 안드로이드 공유 화면으로 전달합니다.
- 선택된 텍스트는 클립보드에 자동으로 복사합니다.
- 대상 앱이 열리면 입력창에 텍스트를 붙여넣으면 됩니다.
- 텍스트만 선택한 경우 공유 화면을 열지 않고 클립보드에 바로 복사합니다.

파일이 여러 개일 때는 다음 방식 중 하나를 선택할 수 있습니다.

- **한번에 공유**: 여러 파일을 원래 형식 그대로 전달합니다.
- **ZIP으로 공유**: 여러 파일을 하나의 ZIP 파일로 묶어 전달합니다.

대상 앱이 다중 첨부를 제대로 지원하지 않는다면 ZIP 공유를 사용하세요.

## 보기 및 설정

- 상단 보기 아이콘으로 카드 보기와 목록 보기를 전환할 수 있습니다.
- **공유받은 뒤 앱 열기**를 끄면 다른 앱에서 공유해도 Share Stack 화면이 나타나지 않고 항목만 쌓입니다.
- **공유 후 스택 비우기**를 켜면 공유를 시작한 뒤 저장된 스택을 비웁니다.
- Android 13 이상에서는 알림과 앱 아이콘의 개수 표시를 위해 알림 권한이 필요합니다.

## 설치

지원 버전은 **Android 8.0(API 26) 이상**입니다.

- [최신 디버그 APK](./share-stack-debug.apk)
- [outputs 폴더의 APK](./outputs/share-stack-debug.apk)

APK를 휴대전화로 내려받아 실행하세요. 브라우저나 파일 관리자에서 설치할 때 기기 설정에 따라 **출처를 알 수 없는 앱 설치 허용**이 필요할 수 있습니다.

현재 제공되는 APK는 개발 및 테스트용 디버그 빌드입니다.

## 안드로이드 제약

- 안드로이드 보안 정책 때문에 일반 앱이 모든 클립보드 복사를 백그라운드에서 계속 감시할 수는 없습니다. 텍스트를 스택에 추가하려면 다른 앱의 공유 메뉴에서 Share Stack을 선택해야 합니다.
- 여러 파일을 받을 수 있는지는 대상 앱의 구현에 따라 달라집니다. 일부 앱은 하나의 파일만 받거나 혼합 파일 형식을 거부할 수 있습니다.
- 이 경우 ZIP 공유를 사용하거나 텍스트를 별도로 붙여넣는 방식이 가장 안정적입니다.
